# 1🚜 Агро-Портал: Система збору первинних даних
Веб-додаток на базі Streamlit, розроблений для автоматизації збору та систематизації агрономічних даних. Система дозволяє заповнювати опитувальні листи, вести облік структури земель та планувати посівні площі з подальшим експортом у формат Excel.

🌟 Основні можливості
📋 Опитувальник агронома: 15 детальних питань про кліматичні умови, технологічні процеси та ресурси господарства.

📊 Структура земель: Інтерактивна таблиця для обліку власних та орендованих угідь (рілля, сінокоси, пасовища).

🌾 Посівні площі: Спеціалізована таблиця для планування культур, врожайності, цін та інтенсивності ЗЗР.

🔗 Синхронізація даних: Поля "Відстань до полів" та "Розмір угідь" автоматично синхронізуються між опитувальником та таблицями.

💾 Розумний експорт: Генерація єдиного Excel-файлу, де кожна сторінка додатка стає окремим листом зі збереженням повного тексту питань.

🧠 Збереження стану: Використання st.session_state дозволяє вільно перемикатися між вкладками без втрати введеної інформації.

🛠 Технологічний стек
Python 3.10+

Streamlit (інтерфейс користувача)

Pandas (обробка табличних даних)

Openpyxl (генерація Excel-звітів)

🚀 Запуск проекту
Встановіть необхідні бібліотеки:

Bash
pip install streamlit pandas openpyxl
Запустіть додаток:

Bash
streamlit run Опитувальник.py
📝 Структура вихідного файлу (Excel)
Після натискання кнопки "Сформувати Excel", ви отримаєте файл з трьома листами:

Анкета: Повний текст питань та ваші відповіді.

Землі: Дані про склад та структуру земельного фонду.

Посіви: Детальна інформація про культури, ЗЗР та послуги підрядників.

⚠️ Важливо
Програма працює локально в браузері. Дані зберігаються в пам'яті поточної сесії. Якщо ви оновите сторінку (F5) або закриєте вкладку браузера, не завантаживши Excel-файл, внесені дані будуть втрачені.
